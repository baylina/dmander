API para agentes

La API reutiliza la misma lógica del wizard web: primero analiza el texto, te dice qué campos faltan y solo publica cuando la demanda ya está completa.

1. Analizar demanda

POST {{ api_base_url }}/api/agent/demands/analyze
Authorization: Bearer TU_TOKEN_API
Content-Type: application/json

{
  "text": "Busco clases particulares de clarinete en Sabadell",
  "known_fields": {
    "level": "principiante"
  }
}

2. Publicar cuando publish_ready = true

POST {{ api_base_url }}/api/agent/demands/publish
Authorization: Bearer TU_TOKEN_API
Content-Type: application/json

{
  "text": "Busco clases particulares de clarinete en Sabadell",
  "known_fields": {
    "level": "principiante",
    "modality": "presencial"
  }
}
